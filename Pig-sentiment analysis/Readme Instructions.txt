Installation and execution steps for Pig sentiment analysis

1> Install linux and set up Hadoop environment.

2>download Pig from
https://pig.apache.org/releases.html

3> If we are using a Virtual Machine, then PIG will be pre-installed.

4> go to the terminal and run PIG grunt shell

pig -x local

This will open pig in our local host.

5> copy the tweets data from the folder, convert it into a TSV file format and place it in your desktop or anywhere in you computer.

6>execute all the commands shown in the program.txt file located in the folder.

7> After executing all the commands, view the output on your desktop.
